# Screenshots

Bilder fürs README und die Doku.

## `dashboard.png`

Screenshot des Betriebs-Dashboards (`/ui`) — Eventstrom-Diagramm über die Zeit,
einklappbares Live-Events-Fenster, Live-Telemetrie und Kennzahlen.

Wird referenziert von:

- [`README.md`](../../README.md) (Cliffhanger oben + Abschnitt „Betriebs-Dashboard")
- [`docs/web-ui-scope.md`](../web-ui-scope.md)

### Aktualisieren

1. `/ui` im Browser öffnen (Token eingeben, **Verbinden**), Dashboard-Tab.
2. Screenshot erstellen und als `docs/screenshots/dashboard.png` ablegen
   (gleicher Dateiname, damit die Doku-Verweise stimmen).
3. Committen:

   ```bash
   git add docs/screenshots/dashboard.png
   git commit -m "docs: Dashboard-Screenshot aktualisiert"
   ```
